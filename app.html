FIZZ BUZZ

Problem Statement

Write a program that prints the numbers from 1 to 100.
  • If the number is a multiple of 3, print "Fizz" instead of that number.
  • If the number is a multiple of 5, print "Buzz" instead of that number.
  • If the number is a multiple of both 3 and 5, print "FizzBuzz" instead of that number.

Enter a FIZZ value and a BUZZ value, then click the button.
